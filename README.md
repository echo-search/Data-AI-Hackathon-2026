# 🌍 Carbon Emissions Intelligence Dashboard
A fully interactive global CO₂ analytics platform (1990–2023)
190+ countries · 33 years of data · Animated visualizations · Trend analysis · Comparisons · Rankings

## Overview
The Carbon Emissions Intelligence Dashboard is an interactive data‑driven platform designed to explore global CO₂ emissions patterns across more than three decades. It combines animated visualizations, country‑level analytics, comparative tools, and dynamic charts to help users understand how emissions evolve across nations, regions, and time.

This dashboard was developed as part of the Data & AI Hackathon 2026, showcasing advanced data visualization, storytelling, and interactive analytics.

## Key Features
- Animated World Map (1990–2023)
- Emissions Race (Bar Chart Race)
- Growth Rate Race
- Dual Country Comparison
- Trend Analysis with moving averages
- Heat Calendar
- Emissions Flow Diagram
- Country Rankings
- Full Data Table
- Carbon Footprint (2007 vs 2023)

## Data Sources
- Global CO₂ emissions dataset (1990–2023)
- Country‑level annual emissions
- Per‑capita and growth metrics
- Regional aggregations
