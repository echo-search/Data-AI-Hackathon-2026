## 🔎 Overview
The **Carbon Emissions Intelligence Dashboard** visualizes global CO₂ emission patterns across 1990–2023.  
It includes animations, comparisons, country rankings, and trend analysis to show how emissions evolve across regions and time.

Developed for the **Data & AI Hackathon 2026**, this dashboard focuses on clear, interactive climate storytelling.

---

## 🚀 Features
- 🌍 Animated World Map (1990–2023)  
- 🏁 Emissions Race (Bar Chart Race)  
- 📊 Growth Rate Race  
- ⚖️ Dual Country Comparison  
- 📉 Trend Analysis (moving averages)  
- 📅 Heat Calendar  
- 🔀 Emissions Flow Diagram  
- 🏆 Country Rankings  
- 📘 Full Data Table  
- 🦶 Carbon Footprint (2007 vs 2023)

---

## 📚 Data Sources
- Global CO₂ emissions dataset (1990–2023)  
- Country-level annual totals  
- Per-capita emissions  
- Growth and regional aggregations

# 📂 How It Works

<details>
<summary><strong>1. Data Loading</strong></summary>
<br>
Datasets in <code>/assets/data</code> are loaded and pre-processed for charts and animations.
</details>

<details>
<summary><strong>2. Visual Rendering</strong></summary>
<br>
Plotly.js renders charts; D3.js handles maps, flows, and animated sequences.
</details>

<details>
<summary><strong>3. User Interactivity</strong></summary>
<br>
Users can explore emissions across 33 years using sliders, dropdowns, and time-based animations.
</details>

<details>
<summary><strong>4. Deployment</strong></summary>
<br>
Hosted using GitHub Pages. Re-deploys automatically on new commits.
</details>
